Introducción a Lit

Lit es una biblioteca de JavaScript moderna para crear aplicaciones web y componentes rápidos, ligeros y reactivos.

¿Y eso qué es?

Un componente web, es una piececita de interfaz gráfica que se pueda reutilizar.

Lit nos facilita el uso de una serie de APIs (Custom elements, Shadow DOM y ) de JS que se pueden utilizar para crear componentes web universales. Es decir, que funcionen en cualquier navegador moderno.

Web Components

¿Y para qué?

Antes de Lit, crear una interfaz gráfica web que fuera compleja llevaba mucho tiempo y esfuerzo.

Por eso se crearon frameworks como Angular, React y Vue.js

La mejora que supone lit, es en el aprendizaje y que luego, al crear componentes "universales" se pueden utilizar también con Angular, React y Vue sin morir en el intento.

El DOM

¿Qué es el DOM?

El Document Object Model (DOM) es una representación del documento (HTML) como un árbol de objetos.

Para interactuar con él se pueden hacer múltiples acciones, entre ellas desatacan...

Seleccionar Elementos


document.getElementById('id');
document.querySelector('.clase');

Modificar Elementos


elemento.textContent = 'Nuevo texto';
elemento.style.color = 'blue';

Crear y Añadir Elementos


let nuevoDiv = document.createElement('div');
document.body.appendChild(nuevoDiv);

Eventos en el DOM


elemento.addEventListener('click', function() {
  alert('Elemento clickado');
});

CSS para decorar

CSS y sus Selectores

Tipos de Selectores en CSS
  • Selector de etiquetas: Selecciona elementos por su nombre de etiqueta.
  • Selector de clase: Selecciona elementos con una clase específica (prefijo .).
  • Selector de ID: Selecciona un elemento único con un ID específico (prefijo #).

Introducción a CSS y HTML

HTML estructura el contenido de la web, mientras que CSS define su presentación visual.

Enlazar CSS a HTML

Usamos la etiqueta <link> para conectar un archivo CSS externo.


<link rel="stylesheet" href="estilos.css">

Selectores y Propiedades

Los selectores definen a qué elementos se aplican los estilos. Las propiedades definen qué estilos aplicar.


/* Aplica texto rojo a todos los párrafos */
p {
  color: red;
}

Estilos en Cascada y Herencia

Los estilos se aplican en orden de especificidad, y algunos estilos se heredan de elementos padres a hijos.

Estilos en Línea vs. Externos

Los estilos en línea se aplican directamente en el HTML, mientras que los externos se mantienen en archivos separados.


<!-- Estilo en línea -->
<p style="color: blue;">Texto azul</p>

Preferencias de Estilo en el Navegador

El navegador sigue la regla de "el último en llegar gana" para estilos en conflicto, priorizando estilos en línea, luego internos y finalmente externos.

Media Queries

Permiten aplicar estilos según las características del dispositivo, como el ancho de la ventana del navegador.


@media (min-width: 600px) {
  body {
    background-color: lightblue;
  }
}

Flexbox y Grid

Sistemas de diseño para crear interfaces complejas y responsivas.


/* Ejemplo de Flexbox */

  display: flex;

}

/* Ejemplo de Grid */

Transiciones y Animaciones

Agrega interactividad visual a tu página con transiciones suaves y animaciones.


/* Ejemplo de Transición */
.hover-me:hover {
  transition: transform 0.5s;
  transform: scale(1.1);
}

/* Ejemplo de Animación */
@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

.animated {
  animation-name: example;
  animation-duration: 4s;
}

Pseudo-clases y Pseudo-elementos

¿Qué son las Pseudo-clases?

Las pseudo-clases son palabras clave que se añaden a los selectores y especifican un estado especial del elemento seleccionado. Por ejemplo, :hover se aplica cuando el usuario señala un elemento.


a:hover {
  color: blue;
}

¿Qué son los Pseudo-elementos?

Los pseudo-elementos permiten estilizar partes específicas de un elemento. Por ejemplo, ::first-line selecciona la primera línea de un párrafo.


p::first-line {
  font-weight: bold;
}

Ejemplos Comunes de Pseudo-clases

:active, :focus, :first-child son algunas de las pseudo-clases más usadas para añadir interactividad y estructura a tu sitio web.


button:focus {
  background-color: lightblue;
}

li:first-child {
  font-size: 120%;
}

Ejemplos Comunes de Pseudo-elementos

::after y ::before se utilizan para insertar contenido antes o después del contenido de un elemento.


p::before {
  content: "Inicio: ";
}

p::after {
  content: " Fin.";
}

Modelo de Caja de CSS

¿Qué es el Modelo de Caja?

En CSS, cada elemento se representa como una caja rectangular, con su contenido, relleno, borde y margen.

Componentes del Modelo de Caja

Cada caja tiene cuatro áreas: contenido, relleno (padding), borde (border) y margen (margin).
